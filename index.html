업무일지 AI 자동 작성기

사용자 양식 기준으로 날짜/매출/중점업무/업무내용/활동계획/특이사항을 기존 엑셀 양식에 반영합니다.

1. 기본 정보

2. 파일 업로드 엑셀 생성 시 필요

매출 파일은 보고용_요약 시트를 읽어서 담당자 기준으로 매출현황에 반영합니다. 김영언 담당자는 세종/대전/대구/혁신점 순서로 정리합니다.

3. 직접 입력 영역

4. 방문 순서 및 출근 인원

기본 방문지는 자동으로 넣지 않습니다. 상담사/큐레이터가 휴무면 상태를 휴무로 선택하면 됩니다.

5. 업무내용 입력

업무유형을 기타로 선택하면 직접 입력 칸이 활성화됩니다. “한 일”과 “후속조치(결과, 기대, 효과 등)” 기준으로 AI가 업무내용을 정리합니다.

6. 생성

아직 생성된 업무내용이 없습니다.
