업무일지 AI 자동 작성기

방문지, 업무내용, 매출 파일을 기준으로 업무일지와 엑셀 파일을 자동 생성합니다.

1. 기본 정보

2. 엑셀 파일 업로드

예: 양식 내용 미작성.xlsx
보고용_요약 시트를 읽어서 담당자/점포 기준으로 정리합니다.

3. 매출 현황

김영언 선택 시 세종점 / 대전점 / 대구점 / 혁신점만 반영합니다.
매출 파일을 업로드하면 여기에 표시됩니다.

4. 중점 추진 업무 / 활동계획 / 특이사항

5. 방문지 / 출근 인원

방문지는 자동 입력하지 않습니다. 필요한 방문지만 직접 추가하세요. 상담사/큐레이터 휴무도 상태로 선택 가능합니다.

6. 업무내용 입력

업무유형이 기타일 때는 기타 업무유형을 직접 입력할 수 있습니다.

7. 생성
